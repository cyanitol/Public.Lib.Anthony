# Contributing

Thank you for your interest.

## Contributions are closed (for now)

At this time we are **not accepting external code contributions** (pull requests,
patches, or direct commits). The repository is open source and tri-licensed for
use, but development is currently maintained internally.

## What you *can* do

- **Open issues** for bugs, feature requests, and questions
- Provide **repro steps**, logs, screenshots, and environment details
- Share **design feedback** and use cases

## Security issues

Please do **not** file sensitive security reports as public issues. Contact the
maintainer privately or follow the process in `SECURITY.md` if present.
