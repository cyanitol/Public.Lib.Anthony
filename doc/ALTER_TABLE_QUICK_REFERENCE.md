# ALTER TABLE Quick Reference

A compact summary of the ALTER TABLE support in Anthony. For details, see
[DDL Features](DDL_FEATURES.md) and [SQL Language](SQL_LANGUAGE.md).

## Supported Forms

```sql
ALTER TABLE t RENAME TO t2;
ALTER TABLE t RENAME COLUMN old TO new;
ALTER TABLE t DROP COLUMN col;
```
