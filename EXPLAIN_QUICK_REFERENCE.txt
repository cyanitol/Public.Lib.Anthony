SYNTAX SUPPORTED
================================================================================

EXPLAIN statement

Examples:

  EXPLAIN QUERY PLAN SELECT * FROM users WHERE age > 18;
  EXPLAIN INSERT INTO users VALUES (1, 'Alice');
  EXPLAIN UPDATE users SET name = 'Bob';
  EXPLAIN DELETE FROM users WHERE id = 1;
